# 醫學(五) 互動測驗（GitHub Pages 版本）

這個資料夾已經是可直接發布到 GitHub Pages 的版本（入口檔為 `index.html`）。

## 一鍵發布（GitHub Pages）
1. 到 GitHub 建立一個新的 repository（public 或 private 都可）。
2. 把本資料夾內的檔案上傳到 repo 根目錄（root）。
3. 進入 repo 的 **Settings → Pages**。
4. 在 **Build and deployment** 選：
   - Source：**Deploy from a branch**
   - Branch：**main** / **(root)**
5. 儲存後，網站網址會是：
   https://<你的帳號>.github.io/<repo 名稱>/

## 特色
- 純前端，題目與答案都在網頁內，不會上傳任何資料。
- 之後要更新題庫，只要替換 `index.html` 再 push 即可。
