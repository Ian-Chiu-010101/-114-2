## 一鍵發布（GitHub Pages）
1. 在 GitHub 建立一個新的 repository（public 或 private 皆可）。
2. 上傳本資料夾內的檔案到 repo 根目錄（root）。
3. repo → **Settings → Pages**
4. **Build and deployment**
   - Source：Deploy from a branch
   - Branch：main / (root)
5. 網址格式：
   `https://<你的帳號>.github.io/<repo 名稱>/`
